# Local Insight Ghana

A static multi-page website prototype for a Ghana tour operator.

## Pages

- Home
- Tours
- Destinations
- Travel Styles
- About
- Responsible Travel
- Travel Guide
- Contact / Plan Your Trip

## Structure

- `images/` - local Ghana tourist-site images
- `styles/` - shared CSS
- `js/` - shared JavaScript interactions

## Preview

Open `index.html` directly or serve the folder with a local static server.
